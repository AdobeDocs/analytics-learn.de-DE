# Schreibgeschützte Projekte in Analysis Workspace

Workspace-Projekte können Benutzern schreibgeschützt freigegeben werden. Wenn ein Empfänger mit der Rolle „Anzeigen“ das freigegebene Projekt öffnet, hat er ein eingeschränktes Projekterlebnis ohne linke Leiste und mit beschränkten Interaktionen.
