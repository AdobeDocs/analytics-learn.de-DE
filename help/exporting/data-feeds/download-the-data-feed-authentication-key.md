# Herunterladen des Authentifizierungsschlüssels für einen Daten-Feed

Durch das Erstellen eines Daten-Feeds erfährt Adobe, wohin Rohdatendateien gesendet werden sollen und was Sie in die einzelnen Dateien aufnehmen möchten. In Analytics ist SFTP-Unterstützung für Daten-Feeds verfügbar. Dazu ist ein SFTP-Host und ein Benutzername erforderlich. Außerdem muss die Ziel-Site einen gültigen öffentlichen RSA- oder DSA-Schlüssel enthalten. Sie können den entsprechenden öffentlichen Schlüssel beim Erstellen des Feeds herunterladen.
