# Herunterladen des Daten-Feed-Authentifizierungsschlüssels

Durch das Erstellen eines Daten-Feeds weiß Adobe, wohin Rohdatendateien gesendet werden sollen und was Sie in die einzelnen Dateien aufnehmen möchten. SFTP-Unterstützung für Daten-Feeds ist in Analytics verfügbar. Dazu müssen ein SFTP-Host, ein Benutzername und die Ziel-Site einen gültigen öffentlichen RSA- oder DSA-Schlüssel enthalten. Sie können den entsprechenden öffentlichen Schlüssel beim Erstellen des Feeds herunterladen.
